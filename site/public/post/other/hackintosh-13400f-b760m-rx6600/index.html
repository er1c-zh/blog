Hackintosh on i5-13400F/B760M/RX6600.

OpenCore i5-13400F/B760M/RX6600 EFI

Feature

  • OpenCore 0.9.2
  • Ventura 13.4
  • All kext release version
  • 传感器 sensor
    • CPU temperature sensor
      • boot-args 增加 lilucpu=17 ，覆盖cpu generation。
    • GPU sensor
      • RadeonSensor.Kext
      • SMCRadeonSensor.Kext
  • 睿频 turbo boost
  • 休眠唤醒
  • iService/AirDrop/etc

Specs

  • CPU i5-13400F
  • MS-终结者 B760M D4 / MAXSUN Terminator B760M D4
  • RX6600
  • Gloway光威 32GB DDR4 3200 台式机内存条 天策系列-皓月白 * 2
  • ZHITAI TiPlus5000 1TB
  • fenvi FV-T919

IMG

wifi && bluetooth

sensor

Usage

  1. 按照OpenCore-Install-Guide构建OpenCore启动盘。

    • 可能需要生成USBMap。
    • 必要时增加一些boot-args输出日志。

  2. luchina-gabriel/BASE-EFI-INTEL-DESKTOP-13THGEN-RAPTOR-LAKE中列出的BIOS设置项。

  3. 进行安装，需要按空格键显示安装的选项。

    • 大概装了半个小时，ETA不准。
    • 会有3次（？）黑屏。

  4. 按照OpenCore-Install-Guide的Post-Install完善安装。

    • 最基本的，挂载EFI，将启动盘中的部分拷贝进去。
    • 生成新的序列号避免问题。

  5. Enjoy it!

MEMO

  • OpenCore的选择器按空格才会展示其他选项。 Some options won’t show unless space pressed.
  • 字体图标渲染发虚，关闭hdr解决。 Disable HDR can solve font blurry for me.
