# BCC

[BPF Compiler Collection](https://github.com/iovisor/bcc)是一个python&&ebpf的封装，提供一个工具集来帮助开发eBPF程序以及一些示例。

# BPF Maps

`BPF Maps`是系统提供的一系列用于在eBPF程序或用户空间程序之间交换数据的数据结构。
